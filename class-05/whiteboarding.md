## Multiple Data Structures
### Why?
- Different usage
    - Array is used to save date in sequnce (order)
    - Dictionary is used to save data to access them by keys
- Performance
```python
    data:
    'name':'Trad'
    'age':23


    Using Arrays:
    ['name','Trad','age',23] 
    Bad performance O(n)


    Using Dictionaries:
    {
        'name':'Trad'
        'age':23 
    } 
    Great performance O(1)
```
